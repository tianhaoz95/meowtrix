Embedded browser

A real browser, in a pane.

Open a browser tab right next to your terminal. A server-side proxy strips frame-blocking headers, so pages that normally refuse to embed render anyway.

  • Preview your dev server beside the shell building it
  • Address bar, back/forward, reload, and open-in-window

What makes it tick

Small, sharp, and built to survive a flaky connection.

🪟

Tiling panes & tabs

Split vertically or horizontally; drag dividers to resize. Each pane holds tabs — a terminal, browser, or code editor.

⚡

Persistent PTYs

Shells outlive the WebSocket. Refresh, switch devices, or hit a network blip — the server replays the buffer and reattaches.

📝

Code editor & Git

Monaco editor with a file tree, side-by-side Markdown/HTML Preview, and a Source Control panel with AI-powered commit messages.

⏰

Scheduled Enter

Queue Enter key presses to fire into terminals automatically (e.g. after quota resets). Lives server-side to survive disconnects.

🐾

On-device Chat Pet

A customizable desktop pet companion (Mochi) powered locally by Google Chrome's built-in Gemini Nano model.

🔥

Keystroke Combo FX

Reward your typing streaks with interactive visuals: particle bursts, screen shake, edge glow, and heat readouts.

🔄

Self-updates

Get notified of updates via in-app banners, and trigger a git-based pull & server restart directly from the workspace.

📱

Mobile-ready

An on-screen key bar with sticky Ctrl/Alt/Cmd modifiers, double-tap autocomplete, and a layout that reflows for touch.

🌐

Embedded browser

A server-side proxy strips frame-blocking headers so otherwise un-embeddable pages render right inside a pane.

🔒

Single active session

Server-coordinated handoff across devices: the newest client claims the workspace; others see a "move session here" overlay.

⌘

Command palette

Press ⌘K (or Ctrl/⌘+Shift+P) for a fuzzy launcher — split panes, open tabs, switch panes, broadcast, themes, settings, schedules, updates.

🛡️

Localhost by default

Binds to 127.0.0.1 so it's off the network until you opt in with --network. Tunnel in over SSH for safe remote access.

🎨

Ten themes

Midnight, Daylight, Ocean, Matrix, Ember, Sakura, and more — terminals are themed from the same CSS variables for free.
